Bitcoin Finance Analyzer Using Apache Spark


Introduction

The bitcoin finance analyzer exploits the use of various Amazon Web Services (AWS) to efficiently analyze Bitcoin market prices. Along with AWS, the CoinDesk API was used to query real-time data and historical data for batch analysis processing using Apache Spark's fast in-memory for computing the analysis.

               

Using the Cryptoanalyzer Effectively

The Cryptoanalyzer focuses exclusively on the price of Bitcoin.

A dashboard has been created that allows you to monitor the price of Bitcoin in real time! Additionally, a 30 day view of bitcoin prices is readily available in the same view.

Furthermore, the web application includes customizable analysis of historical bitcoin data using running averages, exponential averages, and momentum lines. When using this tool, bear with us, as loading may take a while since we are processing very large data sets!
